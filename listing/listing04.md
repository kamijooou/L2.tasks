Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
числа от 0 до 9
deadlock
```
range по каналу будет работать до тех пор, пока канал не закроют. Так как канал в коде не закрыт, то рэндж продолжит работать, и главная горутина повиснет.
